Итак:
1. при открытом окне поиск загрузок (https://platform.trans.eu/exchange/offers) я навожу курсор на поле под надписью Загрузка (Loading) и кликаю на нем левой кнопкой мышки
2. клик левой кнопки мышки под полем из п1 появляется модальное окно с подсказкой внутри поля (Serach)
3. начинаю вводить в поле (Search) модального окна страну в формате ISO-2 (2 буквы) + запятая + пробел + наименование населенного пункта. 
4. при вводе информации в поле из пункта 3 под ним появляется поле с динамически меняющимся списком, фильтрующимся внутренними авлгоритмами  портала транс еу. В открывшемся поле со списком нужно сфокусироваться и нажать на кнопку с максимальной инфорамцией, соответствующей вводимой пользовтелем информации. Например, если нужно задать в фильтре  Loading: CH, 8058, то в открывшемя списке отфильтруются 2 записи: "CH, 8048" и "CH, 8048 Zurich, (Zurich) Zurich district". Нужно выбрать кнопку с максимальной информацией. В нашем примере, "CH, 8048 Zurich, (Zurich) Zurich district".
5. Фокусируюсь и кликаю левой кнопкой мышки на кнопке с максимальной информацией (в примере ("CH, 8048 Zurich, (Zurich) Zurich district). Обратите внимание при наведении на выбранную кнопку появится надпись с посказкой "Additional information". на подсказку нажимать не нужно.
6. после нажатия на кнопку из пункта 5 выпадающий список и модальное  поле (Search) закрываются и в начальном поле (Loading) из пункта 1 зафиксируется инфорация о точке загрузки. В нашм примере будет (CH, 8048, Zurich).
7. точно по такой же схеме п 1-6 выбирается точка Выгрузки (Unloading).
8. Обрати внимание при выборе точки загрузки / выгрузки рядом с полем (Loading) или (Unloading) с указанием (страна + индекс + место) будет дополнительное поле с выпадающим списком (+ 5 km, + 10 km, + 25 km, + 50 km, + 75 km, ...). Это радиус в километрах от выбранной точки, в котором должен производиться поиск мест загрузки. Скрейпер в этом случае по умолчанию должен выбирать значение "+ 75 km". Это же поле с таким же списком до 200км должен быть реализован в нашем фронтенде и приниматься во внимание при обработке фильтров скрейпера,
9. После установки мест загрузки и выгрузки (Loading / Unloading +доп фильтр "+ 75 km", если содержание полей Loading / Unloading содержат информацию "Код страны 2 буквы + запятая + пробел + индекс + проблел + название места". Если указанные поля содержат либо только "Код страны 2 буквы", либо "Код страны 2 буквы + запятая + пробел + индекс"  дополнительное поле с выпадющим списком (+ 5 km, + 10 km, + 25 km, + 50 km, + 75 km, ...) появляться не будет. его искать принудительно не нужно.
10. Если нужно изменить пункты загрузки или выгрузки, то сначала нужно нажать белую кнопку в правом врехнем углу экрана с названием написанным синим шрифтом "EXPAND FILTERS" или "MORE FILTERS". при нажатии на нее откроются поля  с датами, характеристиками машины и т.д. Нас интересуют только даты. Выбор дат должен быть доступен и в нашем фронтенде для обработки скрейпером 
11. После нажатия кнопки "EXPAND FILTERS" или "MORE FILTERS" пользователь выюирает нужные даты
12. Сначала устанавливается Дата загрузки (Loading date), затем Дата выгрузки (Unloading date)
13. Обрати внимание,что полей для выбора каждой даты и даты Загрузки и даты Выгрузки по 2. То есть "Дата Загрузки С" / "Дата выгрузки С"- левое поле, а правое поле напртив него это "Дата загрузки ПО" / "Дата выгрузки ПО"
14. По умолчанию должны быть заполнены оба правых поля ("Дата ... ПО") для Loading date и Unloading date. При этом Дата загрузки (Loading date) должна быть равна текущей дате, а Дата выгрузки (Unloading date) равна следующему рабочему дню. 
15. Выбор даты осуществляется гведением курсора на поле с содержанием DD.MM.YYYY. По клику оевой кнопкой мышки появляется окно с календарем. В календаре выбрать нужную дату = текущей дате. Сначала выбрать дату загрузки, затем выбрать дату выгрузки.
15. После выбора дат в полях зафиксируются значения.
16. Следующим шагом проверяю значение поля Вес, т  ( Weight (t)). которое тоже состит из двух полей: слева (From) и справа (To). Значение правого поля по молчанию должно быть равно "0.9"
17. после установки и проверки всех фильтров пользователь кликает левой кнопкой мышки Зеленую кнопку Поиск ( Search)  в правом верхнем углу экрана
