# Use Case Specification: Notifications

## 1. Введение
Система проактивного информирования пользователей о важных событиях (выгодный груз, проблемы с планом, лимиты).

## 2. Use Cases

### UC-NOTIFY-01: High Profit Cargo Alert (Уведомление о выгодном грузе)
* **Trigger:** `ProfitabilityCalculated` event.
* **Logic:**
    1. Проверить настройки пользователя (порог ставки, например > 0.80 €/km).
    2. Проверить тайм-аут анти-спама (не отправлять чаще 1 раза в 5 мин для этого пользователя).
    3. Сформировать сообщение: "Найден груз! [Откуда] -> [Куда], Ставка: X.XX €/km".
    4. Отправить в Telegram Bot API и Web Push.

### UC-NOTIFY-02: Email Limit Warning (Предупреждение о лимитах Email)
* **Trigger:** Счетчик отправленных писем достиг 40 за последний час.
* **Action:** Отправить системное уведомление диспетчеру: "Внимание! Осталось 10 писем до часового лимита".

### UC-NOTIFY-03: Financial Critical Alert
* **Trigger:** Отставание от плана > 20% (проверка раз в сутки или при обновлении плана).
* **Receiver:** Director.
* **Action:** Email/Telegram уведомление о критическом отклонении.
