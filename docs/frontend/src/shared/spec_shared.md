# Спецификация: Общие UI компоненты и Утилиты (Shared)

**Модуль:** Frontend / Shared
**Родительский документ:** [Architecture - Module 5 & 9]
**Версия:** 1.0

## 1. Обзор
Эта директория содержит переиспользуемые UI компоненты, утилитарные функции и константы, используемые в модулях `fleet`, `loads` и `reports` для обеспечения консистентности дизайна и переиспользования кода.

## 2. Компоненты UI Kit

### 2.1 `StatusBadge`
Используется для отображения рентабельности и статуса транспортного средства.
* **Props:** `type` (success, warning, error, neutral), `label`.
* **Логика (Рентабельность):**
    * 🔴 Красный: < 0.54 €/км
    * ⚪ Серый: 0.54 - 0.59 €/км
    * 🟡 Желтый: 0.60 - 0.79 €/км
    * 🟢 Зеленый: ≥ 0.80 €/км (Логика основана на требованиях приоритетов).

### 2.2 `LocationSelect` (Стиль Trans.eu)
Специализированный инпут для выбора локаций, имитирующий поведение Trans.eu.
* **Поведение:**
    1.  Пользователь нажимает на инпут -> Открывается модалка/выпадающий список.
    2.  Пользователь вводит ISO код (2 символа) + Почтовый индекс + Город.
    3.  Получение подсказок автодополнения через API (Nominatim).
    4.  **Выбор радиуса:** Если выбран конкретный город, показывает выпадающий список "+ km" (+0, +5, ... +200км). По умолчанию +75км для фильтров скрапера.

### 2.3 `OsmMapWrapper`
* **Назначение:** Стандартизированная обертка для карты (Leaflet / React-Leaflet).
* **Возможности:**
    * Принимает маркеры (Старт, Финиш, ТС).
    * Рисует линии маршрутов (пунктирная для порожнего пробега, сплошная для груженого).
    * Рассчитывает и передает данные о дистанции.
* **Действие:** Кнопка "Открыть в новой вкладке" для перенаправления на сайт OpenStreetMap/OSRM.

### 2.4 `SortableTable`
* Универсальная таблица с сортировкой колонок (возрастание/убывание) и пагинацией.
* Используется в списке автопарка и результатах поиска грузов.

## 3. Утилиты (`/utils`)

### 3.1 `geoUtils.ts`
* `calculateDistance(lat1, lon1, lat2, lon2)`: Формула Haversine для быстрой оценки расстояния на клиенте.
* `formatCoordinates(lat, lon)`: Возвращает отформатированную строку для UI.

### 3.2 `currencyUtils.ts`
* `formatEuro(amount)`: Форматирует числа как валюту (например, "1 200.00 €").

### 3.3 `dateUtils.ts`
* Обертки вокруг `date-fns` или `moment`.
* Стандартный формат: `DD.MM.YYYY` (Стандарт Trans.eu).

## 4. Константы (`/constants`)

### `AppColors.ts`
* Определяет семантические цвета для бейджей статусов (Красный/Желтый/Зеленый/Серый) для обеспечения консистентности между Доской грузов и Отчетами.

### `ValidationRules.ts`
* Regex для номерных знаков.
* Мин/Макс значения для габаритов транспортных средств.
