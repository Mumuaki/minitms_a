## 1. Описание
Модуль финансового планирования и контроля. Позволяет устанавливать плановые показатели для транспортных средств и отслеживать их выполнение в реальном времени, а также рассчитывать рентабельность.

## 2. Функциональные требования
* **Планирование:** Установка плана на месяц для каждого ТС (Выручка, Маржа, Пробег).
* **Мониторинг:** Расчет фактического пробега по GPS (Одометр 23:59 - Одометр 00:00).
* **Анализ:** Сравнение План/Факт, расчет средней ставки.
* **Отчетность:** Экспорт данных в Excel.

## 3. Компоненты UI

### 3.1 Planning Table (Таблица планирования)
Интерактивная таблица для ввода планов.
* **Колонки:** Госномер ТС, Водитель, План Выручка (€), План Маржа (€), План Пробег (км).
* **Функционал:**
    * Выбор месяца/года.
    * Inline-редактирование ячеек (для ролей Admin/Director).
    * Автоматический подсчет итогов по всей компании ("Сумма планов по ТС").

### 3.2 Financial Report View (План-Факт анализ)
Детальная таблица выполнения.
* **Строки:** Транспортные средства.
* **Колонки:**
    * ТС.
    * Выручка (План/Факт/Δ).
    * Маржа (План/Факт/Δ).
    * Пробег (План/Факт/Δ).
    * Ставка €/км (Факт).
* **Стилизация:** Ячейки с отклонениями подсвечиваются (Зеленый/Желтый/Красный) согласно бизнес-правилам.

### 3.3 Profitability Calculator (Калькулятор себестоимости)
Вспомогательный компонент (может использоваться в модальном окне или sidebar).
* **Входные данные:** Расстояние (км), Цена (€), Доп. расходы.
* **Расчет:**
    * `Себестоимость = (Расстояние * €/км) + Доп. расходы`.
    * Сценарии (Мин/Сред/Макс рынок).

## 4. Логика и Бизнес-правила
* **Фактический пробег:** Агрегируется с бэкенда, который получает данные от GPS-интеграции.
* **Средняя ставка:** Вычисляется динамически на клиенте или получается с сервера: `Revenue / Distance`.
* **Уведомления:** Если отставание от плана > 20%, генерируется событие для уведомления (отображается в Dashboard).

## 5. Интеграция с API
| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| GET | `/finance/plans` | Получение списка планов на выбранный месяц. |
| POST | `/finance/plans` | Создание/Обновление плана для ТС (Batch update). |
| GET | `/finance/fact-stats` | Получение фактических показателей (агрегация GPS + Orders). |
| GET | `/finance/export` | Скачивание отчета в формате .xlsx. |

## 6. Права доступа
* **Administrator, Director:** Полный доступ (Чтение + Запись/Редактирование планов).
* **Dispatcher, Observer:** Только просмотр (`👁️`).
