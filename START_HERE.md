# ?? START HERE - MiniTMS Server Management

## Добро пожаловать!

Вы находитесь в системе управления сервером MiniTMS. Этот документ поможет вам начать работу за **2 минуты**.

---

## ? Самый быстрый старт (30 секунд)

Откройте PowerShell в этой папке и выполните:

```powershell
.\launcher.ps1
```

Это откроет интерактивное меню со всеми инструментами.

---

## ?? Что делать дальше?

### Вариант 1: Быстрая проверка (рекомендуется)

```powershell
.\check_server.ps1
```

Это займет 30 секунд и покажет статус всех компонентов.

**Результат:**
- ? Зеленые галочки = все работает
- ? Красные крестики = есть проблемы

### Вариант 2: Интерактивное меню

```powershell
.\launcher.ps1
```

Выберите нужную опцию из меню.

### Вариант 3: Прямое подключение к серверу

```powershell
.\connect_server.ps1
```

Получите прямой доступ к командной строке сервера.

## ?? Если что-то не работает

### Проблема: Backend не отвечает

```powershell
.\fix_server.ps1
# Выберите: 1 (Restart Backend Service)
```

### Проблема: Все сломалось

```powershell
.\fix_server.ps1
# Выберите: 5 (Restart All Services)
```

### Проблема: Не знаю что делать

```powershell
.\launcher.ps1
# Выберите: 8 (Quick Start Guide)
```
